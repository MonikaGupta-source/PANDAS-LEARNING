# 🐼 Pandas Learning Journey
 
My personal notes and practice files for learning **Pandas** — from basics to data cleaning, merging, and feature extraction!
 
---
 
## 📚 What I Learned
 
### 📘 Series — `series.ipynb`
- Creating Pandas Series
- Indexing & accessing elements
- Series operations & attributes
 
### 📗 DataFrames — `dataFrames.ipynb`
- Creating DataFrames
- Rows, columns & basic structure
- Accessing data using `df[]`, `df.loc[]`, `df.iloc[]`
 
### 📙 DataFrame Methods — `df-methods.ipynb`
- `.head()`, `.tail()`, `.info()`, `.describe()`
- `.shape`, `.columns`, `.dtypes`
- `.value_counts()`, `.sort_values()`
 
### 📕 Operations — `operations.ipynb`
- Arithmetic operations on DataFrames
- Apply functions using `.apply()`
- Column-wise & row-wise operations
 
### 📓 Data Selection Practice — `data-selection-practice.ipynb`
- Filtering rows with conditions
- Boolean indexing
- Selecting multiple columns
 
### 🧹 Data Cleaning — `data-cleaning-practice.ipynb`
- Handling duplicates — `.drop_duplicates()`
- Renaming columns — `.rename()`
- Changing data types — `.astype()`
 
### ❓ Missing Data — `missing_data.ipynb`
- Finding missing values — `.isnull()`, `.isna()`
- Dropping missing values — `.dropna()`
- Filling missing values — `.fillna()`
 
### 🔗 Merging — `merging.ipynb`
- Merging DataFrames — `pd.merge()`
- Joining — `.join()`
- Concatenating — `pd.concat()`
 
### 📊 Pivot Tables — `pivotTables.ipynb`
- Creating pivot tables — `pd.pivot_table()`
- Aggregating data
- Multi-level indexing
 
### 🔍 Feature Extraction — `feature_Extraction.ipynb`
- Extracting useful features from data
- Creating new columns from existing data
- Real-world data transformations
 
### 🗂️ Dataset Used — `anime.csv`
- Practiced all concepts on real Anime dataset!
